# Smart Irrigation System using Machine Learning

A machine learning-based smart irrigation system developed for farmers to optimize water usage in sugarcane crops.

## Features

* Predicts irrigation requirements using SVM
* Calculates water requirements dynamically
* Uses live weather and forecast API integration
* Multilingual farmer-friendly interface
* Weekly irrigation prediction system

## Technologies Used

* Python
* Flask
* Machine Learning (SVM)
* HTML/CSS
* OpenWeather API
* NumPy

## Purpose

The project aims to improve agricultural efficiency and reduce water wastage through AI-based irrigation recommendations.
